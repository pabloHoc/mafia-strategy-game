# Ideas

This folder contains speculative or experimental concepts.

Ideas explore possible mechanics, themes, and systems that are not yet part of the canonical design.

Ideas may be incomplete, contradictory, or exploratory.

A concept becomes canonical only when moved to `mechanics/`.

For adding a new file in this folder, you must clone the [TEMPLATE.md](TEMPLATE.md) first before editing it.
